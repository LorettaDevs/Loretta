    Source Locations

    Locations of things in source can be represented as 3 different types:

    • TextSpan;
    • LinePosition;
    • and LinePositionSpan.

    TextSpan

    A TextSpan basically represents a range of positions (offset from the start of the @Loretta.CodeAnalysis.Text.SourceText).

    So in the following code:

    print("Hello there")

    The location of the word there could be represented with a TextSpan created with new TextSpan(13, 5) or TextSpan.FromBounds(13, 18).

    A TextSpan also comes with a few builtin methods that allow comparison between instances of it. A few examples are:

    • OverlapsWith(TextSpan): Checks if a TextSpan overlaps with another;
    • IntersectsWith(TextSpan): Checks if a TextSpan intersects with another.
    • Contains(TextSpan): Checks if a TextSpan contains another.

    LinePosition

    A LinePosition is the type of representation of a position in a SourceText that is more suitable for humans to use, as it is a combination of line number and column number.

    Given that is is simply a representation meant for human consumption, it doesn't have any methods other than comparison and equality.

    Obtaining a LinePosition is a bit more of an involved process, requiring using GetLinePosition(Int32) on the Lines of a SourceText.
